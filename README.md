
# Quick Quiz

A simple browser-based quiz app built with HTML, CSS, and JavaScript. Answer questions before the timer runs out — the faster and more consistently you answer, the more points you earn.

## Live Demo

vasanta-quiz-application.netlify.app

## Features

- Score updates instantly as you answer
- 15-second timer for each question
- Immediate feedback on whether your answer was right or wrong
- Bonus points for answering quickly
- Extra bonus points for getting multiple answers right in a row
- Works on both desktop and mobile

## Built With

- HTML
- CSS
- JavaScript (no frameworks)

## How Scoring Works

Each correct answer gives you 10 base points. On top of that:

- You get up to 10 extra points depending on how fast you answered
- If you get two or more answers correct in a row, you get an additional 5 points per question

Wrong answers or running out of time give you 0 points and reset your streak.
